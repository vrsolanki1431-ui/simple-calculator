# 🧮 Simple Calculator

A modern and responsive calculator built using **HTML**, **CSS**, and **JavaScript**. It performs basic arithmetic operations with a clean user interface inspired by Android's Calculator app.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 📊 Modulus (%)
- 🗑️ Clear (C)
- ⌫ Backspace
- 🌙 Dark Mode
- ☀️ Light Mode
- 📱 Responsive Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## ▶️ How to Run


1. Clone the repository

```bash
git clone https://github.com/vrsolanki1431-ui/simple-calculator.git
```

2. Open the project folder.

3. Double-click **index.html** or open it in your browser.
